# Oliver ERP - Portal de Links

Versão com página inicial central em `http://localhost:3000/`.

## Rodar

```powershell
pnpm install
pnpm dev
```

## Telas

- Página inicial: http://localhost:3000/
- Painel Master Beauty: http://localhost:3000/admin
- Cadastro público Beauty: http://localhost:3000/cadastrar-empresa
- Login exemplo Beauty: http://localhost:3000/letsbarbearia/login

## Observação

O Oliver Fit aparece na página inicial como preparação. Quando o módulo Fit real for incluído, os botões serão ligados às rotas dele.
